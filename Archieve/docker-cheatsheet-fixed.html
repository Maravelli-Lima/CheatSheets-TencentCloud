Essential Docker commands and concepts for containerization

Images

  • docker images
    List all images
  • docker pull nginx
    Download image
  • docker build -t app .
    Build from Dockerfile
  • docker rmi image_id
    Remove image
  • docker tag app:latest app:v1
    Tag image
  • docker history app
    Image layers

Containers

  • docker ps
    Running containers
  • docker ps -a
    All containers
  • docker run -d -p 80:80 nginx
    Run detached with port
  • docker stop container_id
    Stop container
  • docker start container_id
    Start container
  • docker rm container_id
    Remove container
  • docker exec -it id bash
    Execute command
  • docker logs container_id
    View logs

Docker Compose

  • docker-compose up -d
    Start services
  • docker-compose down
    Stop services
  • docker-compose ps
    List services
  • docker-compose logs
    View logs
  • docker-compose build
    Build services
  • docker-compose exec web bash
    Execute in service
